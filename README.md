# 📄 Research
## [WikiStar: Shedding Light on the Hidden History of Scientific Wikipedia Articles](https://arxiv.org/abs/XXXX.XXXXX)
**Omer Ehrlich\***, Nitzan Barzilay\*, Rona Aviram, Tom Hope — _EMNLP 2026 System Demonstrations_ &nbsp;<sub>(\*equal contribution)</sub>

An interactive system and LLM benchmark for tracking scientifically meaningful edits across the revision history of scientific Wikipedia articles. We release **WikiStarBench**, an expert-annotated benchmark spanning multiple scientific domains, and evaluate LLMs on classifying scientifically meaningful edits.
